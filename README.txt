Файл: Тестовое задание для прохождение отбора в KATA Academy.
Язык программирования: GO (Golang)
Курс: GO - developer
Разработчик проекта: Саврасов Дмитрий Юрьевич
Возраст: 34

Описание программы:

Консольный калькулятор с возможностью расчета в арабских и римских числах.
Диапазон вводимых значений ограничен для арабских и римских чисел:(-10 <= x <= 10) и (I <= x <= X) соответственно.
Допустимы на ввод только целые числа.
Результат вывода округляется до целых чисел.
Ошибки ввода отработаны согласно техническому заданию.

Лицензия: Свободная (Free)
